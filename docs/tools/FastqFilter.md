## Example
The help menu:
~~~

Usage: FastqFilter [options]

  -l <value> | --log_level <value>
        Level of log information printed. Possible levels: 'debug', 'info', 'warn', 'error'
  -h | --help
        Print usage
  -v | --version
        Print version
  -I <file> | --inputFile <file>
        Path to input file
  -o <file> | --output <file>
        Path to output file
  --idRegex <file>
        Regex to match ID


~~~

To run the tool use:
~~~
biopet tool FastqFilter    
~~~
